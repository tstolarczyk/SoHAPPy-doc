create_csv¶

analysis.population.init.create_csv(file='parameter.yaml', datafile='data.txt', debug=False)[source]¶

From the current paramter file containg the folder to be analysed, create the csv file from default txt file If the paramter file is not given, then the datafile is supposed to contain the full path data file name.

Parameters
fileString, optional

Input parameter file name, can be None. The default is paramter.yaml.

datafileTYPE, optional

DESCRIPTION. The default is “data.txt”.

debugTYPE, optional

DESCRIPTION. The default is False.

Returns
csvfilenameSTRING
None.
