sigmx_after_delay

analysis.population.times.sigmx_after_delay(pop, sigmx_min=5, tmin=<Quantity 1. d>)[source]

Display mean maximal significances reached after a certain time.

Significances are above a minimal value and time ae counted from the explosion. It also displays the fraction of trials reaching the thershold.

Parameters:
popPop instance

Data from disk.

sigmx_minfloat, optional

Mean siginificance value to be considered. The default is 5.

tminAstropy Time, optional

Mean duration to be considered. The default is 1*u.d.

Returns:
None.
