stacked_model

dataset_tools.stacked_model(ds, ds_stack=None, first=False, debug=False)[source]

This function is not finalised and should be checked. It is intended to recompute an effective spectral model from the exiisting already stacked model -from the previous call- and the current model in a dataset list. It also extract the masked dataset of the first dataset in the list (when first == True).

The reconstructed binning is required to apply the masking The theoretical spectrum is therefore evaluated on reconstrcuted energies which assumes that the reconstructed energy is not too different from the true one.

Parameters
dsDataset

The current dataset.

ds_stackDataset, optional

The current stacked dataset if first is False. The default is None.

firstBoolean, optional

If True, extract the masked dataset - Valid for the first dataset of the list. The default is False.

debugBoolean, optional

If True, let’s talk a bit. The default is False.

Returns
modelSkymodel

The current stcake model.
