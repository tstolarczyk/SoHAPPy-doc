Minimal acceptable True (i.e. generated) energies depending on the IRF zenith angle. Note: The masking later will remove the bin containing a given E value. If the E value is an edge, the subsequent bin is lost. The minimal and maximal energies need therefore to be slighlty before, resp. after the first, resp last edge of interest.
