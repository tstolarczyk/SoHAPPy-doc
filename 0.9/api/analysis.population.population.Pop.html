negative_significance()[source]

Negative significances The significance is returned by the WStatCountsStatistic function in gammapy, and has a sign which corresponds to the sign of the excess. As a consequence it can be negative. In case the simulation is done without fluctuation and one trial, the excess can not be negative, and all significances are zero (no excess count) or more.
